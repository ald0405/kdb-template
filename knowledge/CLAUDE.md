# Knowledge Files

This folder contains reference files for KDB. Load the specific file you need — do not load all files by default.

## File index

| File | What it contains | When to load |
|------|-----------------|--------------|
| `prd-template.md` | PRD structure with section guidance | Any PRD drafting or review work |
| `product-glossary.md` | Metric definitions, product architecture, funnel stages, customer segments | Before answering any metric question or discussing product concepts |
| `okrs-and-sprint-goals.md` | Current OKRs, sprint goals, initiative summaries | Before making any product recommendation or prioritisation call |
| `slack-channels.md` | Channel names, purposes, audiences, and posting rules | Before routing any Slack message |
| `dashboard-index.md` | Looker and Amplitude dashboard IDs, URLs, query recipes, filter patterns | Before querying Looker or Amplitude |
| `team.md` | Team roster, roles, Slack handles, Jira usernames, sign-off matrix | Before assigning Jira tickets, tagging people in Slack, or writing targeted comms |
| `people/{slug}.md` | Distilled stakeholder intelligence — patterns, question types, decision style | Before any meeting with that person, or when writing comms targeted at them |
